# Mandatory Skills Usage - Antigravity

## Critical Rule

**BEFORE any implementation, design, or problem-solving work:**

You MUST invoke the appropriate skill. This is not optional.

## Skill Invocation Requirements

### 1. Brainstorming (MANDATORY)
**Invoke BEFORE:**
- Any new feature implementation
- Any architectural decision
- Any design work
- Any creative problem-solving

**How to invoke in Antigravity:**
```python
view_file("~/.gemini/antigravity/skills/brainstorming/SKILL.md")
```

**Never skip this.** Even "simple" tasks benefit from structured thinking.

### 2. Test-Driven Development (MANDATORY)

**How to invoke in Antigravity:**

### 3. Systematic Debugging (MANDATORY)
**Invoke WHEN:**

**Find root cause first.** No quick fixes.
