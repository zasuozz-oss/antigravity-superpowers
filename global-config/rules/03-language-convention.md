# LANGUAGE CONVENTION

## Default Language: English

All output should follow these language rules:

| Content Type | Language |
|-------------|----------|
| Code, variables, functions, classes | English (always) |
| Comments, logs, error messages | English |
| Commit messages, PR descriptions | English |
| Documentation, README, guides | English |
| User-facing explanations, responses | English |

## Override Rules

- **Always respond in the default language**, regardless of the language of the user's input.
- Only switch language if the user **explicitly requests** a different language (e.g., "respond in Vietnamese").
- Code identifiers (variables, functions, classes) must ALWAYS be in English regardless of conversation language.
