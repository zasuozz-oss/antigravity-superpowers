# Add Workflow

## Purpose
- Introduce new functionality or systems with clear scope
- Follow script-folder skill rules

---

## Steps

1. **Clarify Scope and Requirements**
   - Understand what the user wants
   - Ask max 3 short questions if critical info missing

2. **Propose Class Responsibilities**
   - Single responsibility per class
   - Clear naming

3. **Design Data Models and Flow**
   - Data structures
   - State flow
   - Dependencies

4. **Implement Code**
   - Follow csharp-conventions skill
   - Follow unity-mobile skill
   - Minimal, focused implementation

5. **Explain Usage and Validation Steps**
   - How to use the new feature
   - How to verify it works

---

## Constraints
- Keep implementation minimal and maintainable
- Do NOT add features outside the request
- Do NOT introduce new frameworks unless requested
