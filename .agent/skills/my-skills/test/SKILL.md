# Test Workflow

## Purpose
- Run unit tests and report results
- Analyze test coverage and gaps
- Generate test scaffolding for scripts

---

## Modes

### test run
- Execute tests and report results
- Show passed/failed/skipped counts
- Show failure details

### test analyze
- Analyze existing test coverage
- Identify gaps in test coverage
- Suggest areas needing tests

### test generate
- Create test scaffolding for a target script
- Follow Unity Test Framework (NUnit-based)
- Generate test class with basic structure

---

## Constraints
- Do NOT modify production code during test runs
- Follow existing project test patterns
- Prefer Unity Test Framework (NUnit-based)
