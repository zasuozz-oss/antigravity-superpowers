# Document Workflow

## Purpose
- Generate documentation for scripts, features, or folders
- Support inline XML docs or markdown format

---

## Modes

### document script
- Add inline XML comments to public APIs
- Or create separate .md file for script documentation

### document feature
- Create high-level feature documentation
- Explain purpose, flow, usage

### document folder
- Generate README.md for a module/folder
- List contents and responsibilities

---

## Constraints
- Do NOT modify code logic
- Do NOT refactor or rename
- Use Vietnamese for descriptions (per 00-ai-rules.md)
